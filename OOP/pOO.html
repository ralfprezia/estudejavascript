JAVASCRIPT

Herança em JavaScript

  • Baseada em protótipos

      • prototype

    • É uma variável que armazena as definições do nosso objeto

      • _proto_

    • É uma variável de referência que aponta para o prototipe do tipo criado


      • Este tipo é chamado de:

      constructor

      Baseado em um constructor é criado um prototype e nesta varaiável
      é armazenada as refêrencias pelo _proto_!

Classes

  • A classe é uma herança do prototype.

      • ES6

    • Foi criado no ES6
    • É uma simplificação da herança de protótipos
    • Sempre usado com a palavra chave class.

      • É uma super sintaxe das funções!

Modificadores de Acesso

  • Não possui modificador de acesso
  • Básicamente controla o que é privado dentro da classe.

    • É realizado no NodeJs, mas ainda não tem suporte nos browsers.Data(16/04/2021)

Encapsulamento

  • É basicamente ocultar detalhes do funcionamento interno.

Static

  • Permite métodos/atributos sem instanciar uma determinada classe.
